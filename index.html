¿ Quíenes Somos ?

Este proyecto se realiza con fines académicos, el objetivo es demostrar las competencias adquiridas a medida que se avanza en el proceso de formación para alcanzar el título de técnico profesional en programación de sistemas de información. El contenido de este trabajo está fundamentado en la debida investigación de la crianza de porcinos, como también en la consulta y uso de las herramientas necesarias para su debido desarrollo. Con este software daremos alcance a las necesidades del cliente granja porcícola Wilbur, permitiendo minimizar los procesos para la crianza de cerdos y maximizando la productividad de la granja.
